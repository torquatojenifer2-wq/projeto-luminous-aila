🚀 Agente AILA - Triagem Inteligente de Chamados (Luminus)Este projeto é um Produto Mínimo Viável (MVP) desenvolvido para a Luminus Energia & Engenharia, com foco na automação de Ordens de Serviço (OS) do sistema AUVO. O objetivo principal é reduzir em 20% as visitas técnicas desnecessárias através de uma triagem inteligente baseada em Machine Learning.🛠️ Tecnologias UtilizadasPython 3.11 (Flask): Core da inteligência e API de triagem.Scikit-learn & Pandas: Processamento de dados e modelo de classificação de severidade.Docker & Docker Compose: Arquitetura de microserviços e isolamento de ambiente.n8n: Orquestrador central integrado à API do AUVO.Lovable: Interface PWA para monitoramento em tempo real.🏗️ Arquitetura da SoluçãoA solução foi desenhada seguindo a metodologia Top Hawks, focando em escalabilidade e impacto operacional.Entrada: O chamado técnico é registrado no AUVO.Orquestração: O n8n detecta a nova OS e envia o texto para este Agente AILA.Inteligência: O script Python processa o chamado e classifica em:Low: Sugestão de Resolução Remota.Medium: Sugestão de Visita Agendada.High: Sugestão de Visita Emergencial.Saída: O n8n atualiza o status no AUVO e notifica os envolvidos via WhatsApp.🚀 Como ExecutarGraças à conteinerização com Docker, o deploy é simplificado e resiliente.Certifique-se de ter o Docker Desktop instalado.Na raiz do projeto, execute:Bashdocker-compose up --build -d
A API estará disponível em http://localhost:5000/triage.📊 Impacto de NegócioOtimização de Tempo: Respostas mais rápidas e assertivas diretamente no AUVO.Redução de Custos: Menos deslocamentos técnicos para problemas que podem ser resolvidos remotamente.Segurança Operacional: Sistema com reinicialização automática em caso de falhas via Docker.
